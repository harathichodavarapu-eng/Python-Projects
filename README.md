## Overview

This repository contains all the internship tasks completed as part of the assignment. The tasks include:

1. **API Integration and Data Visualization**  
   - Fetches data from a public API (OpenWeatherMap).  
   - Visualizes weather data using Matplotlib.  
   - **Files:** `Task1/task1_api.py`, `Task1/task1_chart.png`

2. **Automated Report Generation**  
   - Reads data from a file and generates a PDF report.  
   - **Files:** `Task2/task2_report.py`, `Task2/task2_report.pdf`

3. **AI Chatbot with NLP**  
   - Simple chatbot capable of answering user queries using Python.  
   - **Files:** `Task3/task3_chatbot.py`

4. **Machine Learning Model Implementation**  
   - Implements a predictive model using Scikit-learn.  
   - Classifies outcomes from a sample dataset (Iris dataset).  
   - **Files:** `Task4/task4_ml.py`
